### How to Run?

- Make sure to have installed java and apache maven

```bash
java --version
```
```bash
mvn --version
```
- Into directory
```bash
cd pepechess
```
- Compile with maven
```bash
mvn compile package
```
- Run the game
```bash
java -jar ./target/chess_game-1.0-SNAPSHOT.jar
```
